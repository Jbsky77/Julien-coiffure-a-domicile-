## Architecture multi-entreprises

- Chaque donnée métier est rattachée à une entreprise.
- Le super-administrateur pilote les entreprises et les abonnements sans appartenir à leurs équipes.
- L’accès d’assistance à une entreprise est explicite et journalisé.
- Le nom et le logo affichés proviennent de l’entreprise active.

> Déploiement multi-entreprises vérifié depuis la branche principale.
